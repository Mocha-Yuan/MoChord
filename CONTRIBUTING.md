# Contributing

Thanks for your interest in improving MoChord.

## Development Setup

Install dependencies:

```bash
npm install
```

Run the web app:

```bash
npm run dev
```

Run the desktop app:

```bash
npm run desktop:dev
```

Build before opening a pull request:

```bash
npm run build
```

## Environment Variables

Copy `.env.example` to `.env` if you want to test DeepSeek features in the desktop app.

Never commit `.env` or any real API key.

## Pull Requests

- Keep changes focused and easy to review.
- Update README or related docs when behavior changes.
- Include screenshots or short recordings for visible UI changes.
- Mention any tests or build commands you ran.
